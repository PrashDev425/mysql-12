# Software Installation

## XAMPP

**XAMPP** is a ``free``, ``open-source software`` package that provides a complete local web server environment for developing and testing web applications. It includes **Apache (web server)**, **MySQL/MariaDB (database server)**, **PHP (server-side scripting language)**, and **Perl**.

**XAMPP** Installation and Setup 

- Download and install **XAMPP** on your computer. [https://www.apachefriends.org/]
- Open the **XAMPP Control Panel**.
- Click Start next to ``Apache`` and ``MySQL`` to run both services.
- To manage the database, click the ``Admin`` button next to ``MySQL`` or open:
    ```
    http://localhost/phpmyadmin
    ```
